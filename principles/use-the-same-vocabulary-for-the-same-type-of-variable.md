Bad:
```js
getUserInfo();
getClientData();
getCustomerRecord();
getPersonDateOfBirth();
```
Good:
```js
getUser();
getUserDateOfBirth();
```

## Why

* Using the same vocabulary creates clarity and understanding
* Reduces confusion
